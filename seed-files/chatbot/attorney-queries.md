# Attorney Mode — Sample Queries

Use these with the chat set to **Attorney** mode. Each scenario includes an opening question and follow-ups to demonstrate multi-turn conversation (max 5 turns). Attorney mode produces more detailed analysis with statutory citations and case law.

---

## Scenario 1: FEHA Burden-Shifting Analysis

**Query 1:**
> Analyze the burden-shifting framework under FEHA for a disability discrimination claim where the employer claims the employee was terminated for poor performance documented over 6 months.

**Follow-up 2:**
> What is the standard for establishing pretext when there is a documented performance history?

**Follow-up 3:**
> How does the interactive process obligation factor into the analysis if the employer never engaged in it?

**Follow-up 4:**
> What damages are available if we prevail on both the discrimination and failure to engage in the interactive process claims?

**What it demos:** Deep statutory analysis, McDonnell Douglas framework, FEHA-specific burden shifting, damages breakdown. Shows tl;dr + Short Answer + Analysis format with citations.
